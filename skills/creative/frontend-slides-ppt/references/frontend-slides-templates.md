# Frontend Slides 34套模板参考

来源: zarazhangrui/frontend-slides (⭐20,500)
仓库: https://github.com/zarazhangrui/frontend-slides

## 模板分类

### 封面型模板
大字标题+干净留白，适合产品发布、项目启动、投资人pitch
- 大标题 + 副标题 + CTA按钮
- 全屏视觉 + 居中文字
- 品牌标志 + 关键信息

### 内容型模板
信息模块化展示
- 漏斗图: 转化率/流程展示
- 流程图: 步骤/路线展示
- 对比表: A方案 vs B方案
- 增长曲线: 趋势/数据展示
- 时间线: 里程碑/历史节点
- 矩阵图: 竞品/策略对比

### 数据型模板
- KPI卡片: 大数字+小标题
- 数据图表: 柱状图/饼图/折线图
- 财报摘要: 收入/利润/增长

### 技术型模板
- 架构图: 系统层级/模块依赖
- API文档: 接口/参数/示例
- 代码演示: 关键代码+运行结果

### 品牌型模板
- 企业介绍: 公司/使命/团队
- 案例研究: 问题/方案/结果
- 品牌故事: 历程/价值观/愿景

## 使用方式
1. `git clone` 仓库获取真实HTML模板文件
2. 每个模板是独立HTML文件，可修改颜色变量/内容占位符
3. 所有模板无外部依赖，单个HTML可直接浏览器打开
