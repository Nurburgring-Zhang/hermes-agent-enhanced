## 问题5维分类

### Ⅰ 维度组合冲突（核心，占80%）

这是**维度的排列组合灾难**——17维片段各自合法，但组合在一起产生物理矛盾。

#### Ⅰ-A: A01+A02 族裔+发色冲突
- 亚裔+金发 / 铂金 / 银白发 / 亚麻色 / 紫发 / 绿发 / 蓝发 / 粉发 / 红发 / 挑染 / 渐变
- 根源：A02维度库中有约5%的非天然发色片段，A01选了亚裔后没有过滤干净

#### Ⅰ-A2: A01+A04 族裔+瞳色冲突
- 亚裔+蓝眼 / 绿眼 / 碧眼 / 紫眼 / 灰眼 / 金眼 / 琥珀眼 / 红眼
- 根源：A04维度库中有大量蓝眼/绿眼片段（来自非亚裔数据）

#### Ⅰ-B: A01+D01 性别+服装冲突
- 男子+裙子/连衣裙/旗袍/蕾丝/胸罩/比基尼/丝袜/高跟鞋/头纱/芭蕾舞鞋/蓬蓬裙
- 根源：D01片段没有性别标签，男性也能取到女性化服装

#### Ⅰ-C: B01+B01 场景冲突
- 室内+户外自然、卧室+雪山、浴室+森林、客厅+海滩
- **解决方案：B01只取1个（v9最大改进）**

#### Ⅰ-D: B01+D01 场景+服装不匹配
- 办公+比基尼、教室+泳装、医院+晚礼服
- 根源：维度之间没有做风格一致性检查

#### Ⅰ-E: A01+A05 婴儿/儿童+成人化
- 婴儿+比基尼/内衣/蕾丝/丝袜/瑜伽/下犬式
- 根源：婴儿的A01片段和成人的A05/D01片段组合

### Ⅱ API扩写失控（5%）

这部分是API自身创作行为，后处理无法完全拦截：
- 动物拟人未说明（狗穿衣服有头发、猫穿运动内衣）
- 血腥暴力（沾满鲜血的手）
- 穿两双鞋（芭蕾舞鞋+踝靴）
- 超现实不合理（湖底+餐桌共存、洞穴+沙发）

### Ⅲ 风格开篇异常（7.5%）

- "线风"、"代风"、"度风"、"深风"、"焦风"——截断为2个字的风格名
- "风风"——重复
- "，风"——前面是逗号
- 根源：后处理补风格时检测不严

### Ⅳ 后处理漏检（5%）

- 裸体+服装共存（检测模式不够精准）
- 三手（左手+右手+双手同时出现）
- 地板类型矛盾（木地板+草地、地毯+沙地）

### Ⅴ 维度库数据缺陷（2.5%）

- B01片段自身含混合场景（"床边有窗外森林"）
- 这是源头问题，41条规则无法完全解决
