## 九层模板（单行条目，580-730字符）

每一条为**一个完整的自然段**，不分段、无标题、无序号。严格包含以下9层，按顺序自然融合：

| 层 | 内容 | 说明 |
|----|------|------|
| 1 | **场景设定**（1-2句） | 城市+具体地点+天气/季节+光线氛围。开篇即场景，如"一条瑞士圣莫里茨湖面冰封的冬季雪山时尚大片" |
| 2 | **主体穿搭**（2-3句） | "主体是一位穿XX的女性在XX位置做XX动作"，含服装面料（真丝/羊绒/丝绒/精纺羊毛）、款式细节（深V/立领/裙摆开衩/茧型/斜裁）、配饰（披肩/手套等） |
| 3 | **珠宝首饰**（1-2句） | 颈间/耳畔/指间/手腕的项链、耳坠、戒指、手镯。含材质（铂金/黄金/白金）、宝石（红宝石/蓝宝石/翡翠/钻石/珍珠/碧玺/青金石/琥珀）、切割方式（枕形/梨形/祖母绿切割/水滴）、颜色（鸽血红/矢车菊蓝/冰种飘绿/蜜糖色/南洋金珠） |
| 4 | **传统色彩分析**（2-3句） | 用"传统色彩以XX为基调——"引出，列出3-5种传统色名。必须使用中国传统色名体系（朱砂红/胭脂虫红/松花绿/缥碧色/绀青/鸦青/月白/石青/象牙白/焦糖棕/栗壳色等） |
| 5 | **相机与镜头**（1句） | 具体品牌型号（哈苏H6D/徕卡SL2/富士GFX50S/徕卡Q2/徕卡M10-P/徕卡M11）+ 焦距（28/35/50/85/110mm）。用"采用...的XX风格摄影/时尚摄影/街拍大片"格式引导 |
| 6 | **光线技术**（1-2句） | 主光源（自然光/人造光/混合光）、方向（侧逆光/侧光/顶光/漫射光）、性质（硬光/柔光/金色暖光/冷白柔光）、辅助光（雪地/冰面/水面/地面/白色建筑反射补光）、光比描述（一比三/四档宽容度等） |
| 7 | **构图方法**（1-2句） | 构图类型（对称/三分法/黄金分割/对角线/S形/框景）、人物位置（右侧/左侧/中央/下方/左上）、引导线来源（桥钢缆/屋檐/栏杆/石柱/地毯楼梯）、景深层次（前景/中景/远景） |
| 8 | **空气氛围**（2-3句） | "空气里是..."格式：气味（草木/海水/咖啡/香水/木材/雪茄/茉莉/花椒麻）、温度感觉（零下冷空气/干燥热风/湿润晚风）、声音（钟声/雨声/海鸥/爵士乐/剪刀裁布/掏耳朵铁钳/交通嗡鸣/宣礼回响） |
| 9 | **情感收尾**（1句） | "氛围是..."格式：提炼整体氛围（阿尔卑斯雪国意志/北京雨夜静谧力量/东京金色光雾瞬间/英伦凌厉优雅/神圣合一永恒/都市女性力量感/胶片电影凝固/东方水墨韵致） |

## 字符约束

- **每条严格 580-730 中文汉字**
- 用 `python3 -c "print(len(open('file').readline()))"` 验证
- 低于580：补充珠宝细节、光线技术参数、空气氛围感官
- 高于730：压缩色彩分析和场景设定部分，合并并列小句
