## 11套初始模板
1. bold-signal — 大色块+大标题, 产品发布/投资人pitch
2. terminal-green — 终端黑暗风, CLI教程/黑客风/技术演示
3. newsroom — 报纸特稿, 热点解读/深度报道
4. electric-studio — 白底电光蓝, B2B演讲/投资人路演
5. bauhaus-bold — 包豪斯+布鲁塔利, 观点宣言/品牌主张
6. creative-voltage — 创意工作室, 设计周/作品集
7. neon-cyber — 霓虹赛博, AI/大模型/web3/科技
8. vintage-editorial — 复古专栏, 个人观点/文化随笔
9. split-canvas — 左右双色画布, 双主题对比/过去vs现在
10. dark-botanical — 自然博物馆, 品牌故事/时尚/美妆/旅行
11. forest-ink — 旧版国家地理, 自然/户外/纪录片/人文

## 11套新增模板（2026年6月更新）
12. hero-product — 加强版产品Hero, 适合品牌大片
13. data-chart-nyt — NYT风格数据图表, 数据叙事
14. glitch-title — 故障艺术标题, 开场/系统主题
15. liquid-bg-hero — 流体渐变Hero, 产品揭示/大胆声明
16. light-leak-cinema — 暖色胶片光晕, 品牌影片/情绪叙事
17. typewriter-cursor — 打字机光标, 代码风格揭示/CLI演示
18. logo-outro — 动画Logo结束卡, 品牌签章/视频结尾
19. kinetic-type — 动态排版, 音乐/艺术/创意
20. swiss-grid — 瑞士网格数据卡片, 简洁信息展示
21. vignette-data — 暗角数据卡片, 数据故事/统计分析
22. takram-organic — Takram有机动效, 自然/科技交叉

## 推荐主题-内容匹配
- 产品发布/投资人pitch → hero-product / bold-signal
- 技术教程/CLI演示 → terminal-green / typewriter-cursor
- 数据汇报/分析报告 → data-chart-nyt / vignette-data
- 品牌宣传/故事 → dark-botanical / light-leak-cinema
- 观点/文化/艺术 → kinetic-type / vintage-editorial
- 对比/辩论 → split-canvas
- 视频结尾 → logo-outro
