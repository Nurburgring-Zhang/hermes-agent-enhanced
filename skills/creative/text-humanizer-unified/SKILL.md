# Unified Text Humanizer

合并自: humanizer (30KB, 英文) + ai-text-humanizer (2KB, 中文)

## 语言自动检测
- 输入含中文字符 → 中文模式
- 纯英文 → 英文模式

## 中文模式
AI文本润色规则：打破句式对称、增加口语化、增强情绪感染、具象化表达
- 所有AI生成内容的最后一步必用

## 英文模式
Strip AI-isms and add real voice.
- Sentence rhythm breaking
- Authenticity injection
- Emotional tone adjustment
