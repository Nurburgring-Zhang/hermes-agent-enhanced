# 热点雷达 (Hot Trend Radar)

## 热点分类

### 常规可预测热点
| 类型 | 示例 | 准备方式 |
|------|------|---------|
| 行业展会 | CES、世界人工智能大会 | 提前准备框架稿 |
| 固定节点 | 春节、双11、618 | 提前准备模板 |
| 财报季 | 公司季度/年度财报 | 提前整理数据模板 |
| 政策发布 | 两会、行业政策 | 关注发布日程 |

### 突发不可预测热点
| 类型 | 示例 | 响应方式 |
|------|------|---------|
| 重大新闻 | 科技突破、公司收购 | 2小时内出稿 |
| 争议事件 | 公司CEO言论、产品事故 | 事实先行+谨慎定位 |
| 行业地震 | 监管突变、技术被禁 | 快速分析+价值导向 |

## 蹭热点公式

```
热点事件 + 自身领域 + 独特视角 = 爆款选题

示例：
热点: #董明珠称不用女性员工#
选题: 《董明珠这句话，暴露了职场最深的偏见》

热点: #AI取代程序员#
选题: 《AI取代的不是程序员，是不懂AI的程序员》
```
