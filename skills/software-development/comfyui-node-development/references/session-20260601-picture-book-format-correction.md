# 2026-06-01 儿童绘本格式修正 — 与分镜格式彻底区分

## 问题

儿童绘本格式（_build_child_book）之前被错误地使用了分镜风格的输出字段：
- 时间·空间锚定（分镜用，绘本不需要）
- 分镜场景/角色特征（分镜用，绘本不需要）

导致AI输出把角色变化塞进角色特征字段（实际应属于画面描述），把时间变化塞进分镜场景字段。

## 正确的儿童绘本输出格式

参考用户提供的《鲸鱼科普》绘本示例：

```
第1页：封面
画面：巨大的蓝鲸横跨跨页，渺小的潜水员在旁。
文字：嘘，别出声，我们正在拜访海洋里最大的巨无霸。

第2页：体型对比
画面：蓝鲸背上停着三辆大校车。
文字：蓝鲸有多大？它和三辆大校车连起来一样长！
```

### 格式要点
- **第X页：标题** — 每页必须有标题行
- **画面：** — 画面描述，2-3句，单页只写一个凝固瞬间动作
- **文字：** — 1-2行，每行不超过15字，适合儿童朗读
- **场景变化/角色变化** — 仅在有大变化时在文字段后加一行描述，无变化时不出现

### 格式与传统绘本对齐
绘本格式不应该包含：
- ❌ 时间·空间锚定（那是分镜用的）
- ❌ 分镜场景字段（那是分镜用的）
- ❌ 角色特征字段（那是分镜用的）
- ❌ 动态描述【动态】（那是视频格式用的）

绘本格式应该参考：
- 儿童科普绘本（如鲸鱼科普）——每页独立画面+标题+文字
- 传统故事绘本——每页一个场景+文案

## 代码修改位置

`_build_child_book()` 在 `/mnt/d/ComfyUI/custom_nodes/PromptLibraryNode/__init__.py` 中。

修改前：
```
输出格式
以第X页开头，每页包含：时间·空间锚定...、画面...、文字...、分镜场景...、角色特征...
```

修改后：
```
输出格式
第X页：标题
画面：2-3句，单页只写一个凝固瞬间动作
文字：1-2行，每行不超过15字
（场景变化时加一行：场景变化：完整场景描述）
（角色变化时加一行：角色变化：完整角色描述）
```
