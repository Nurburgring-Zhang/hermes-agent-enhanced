# 故事感总纲 — Story Sense Framework

## 可直接嵌入system prompt的剧本理论系统

来源：全球8大顶级剧本理论体系融合（皮克斯22条、英雄之旅、三幕剧、救猫咪15节拍、Dan Harmon故事圆、情感弧线理论、悬念与钩子理论、儿童故事特殊性）

## 核心原则

你欣赏一个角色的努力拼搏，而不是他本身。故事是角色在克服困难中展现的真实成长轨迹。每个故事的核心只有一个——角色想要某样东西，但遇到障碍。

## A. 核心叙事结构（六段式弧线）

### 第一段：开场钩子（前10%-15%）
目标：3秒内建立好奇/悬念/情感连接
- 反直觉视觉：展示一个打破预期的画面
- 疑问式旁白：用一个问题开场
- 情感抓取：直接展示角色的强烈情绪
- 悬念物体：展示一个神秘物品
- 角色反差：身份与环境冲突
规则：开场必须包含「角色+愿望+独特情境」三要素。

### 第二段：情感建置（15%-30%）
目标：让观众爱上/关心角色
- 展示角色的愿望（具体、可见、可共情的小目标）
- 展示角色的独特性（一个可爱的小毛病/习惯/弱点）
- 建立情感锚点（口头禅/特殊动作/标志性物品）
规则：不用旁白解释角色，用行动定义角色。

### 第三段：冲突升级（30%-60%）
目标：压迫感逐步增强，张力螺旋上升
- 3波阻碍原则：难度递增的三次阻碍
- 第一波：小阻碍（容易解决，但消耗了资源/信心）
- 第二波：中阻碍（需要改变策略/寻求帮助）
- 第三波：大阻碍（看似无法逾越，核心信念受挑战）
规则：每一波都比上一波更个人化。

### 第四段：转折点/灵魂黑夜（60%-75%）
目标：最低谷→新的希望
- 「一无所有」时刻：角色失去最重要的东西
- 情感最低点：观众最心疼角色的时刻
- 自我质疑：角色问自己「我是不是不够好」
规则：转折不能靠巧合，必须来自角色之前行为/关系的自然延伸。

### 第五段：高潮解决方案（75%-90%）
目标：情感释放+视觉冲击
- 角色用学到的新方法解决问题
- 视觉高潮：画面/节奏的峰值
- 所有伏笔在此回收

### 第六段：情感闭环（90%-100%）
目标：角色成长+关系升华+结尾呼应开场
- 呼应开头：开场的问题在结尾得到答案
- 价值观闭环：开场问题=结尾答案
- 留10%让观众自己想象

## B. 情感曲线规则

标准情感形状（波动上升）：
```
开场（好奇）→建置（微开心）→第一阻碍（微挫折）
→小胜利（微回升）→第二阻碍（明显挫折）→再尝试
→第三阻碍/最低点→希望出现→高潮（最高点）
→结尾（温暖满足）
```

情感可视化规则：
- 表情变化：从笑到哭到笑的过程必须画出来
- 身体语言：低头→抬头→蹦跳，蜷缩→伸展→奔跑
- 环境色彩：暖色→冷色→暖色（更明亮）
- 声音节奏：快→慢→快
- 大小对比：变小→变大

禁止：直线上升、突然反转、平铺直叙、情感断层。

## C. 钩子设计规则

### 开场钩子类型
| 类型 | 方法 | 示例 |
|------|------|------|
| 反直觉视觉 | 打破预期的画面 | 一只猫在给老鼠盖被子 |
| 疑问式旁白 | 用问题开场 | 你知道世界上最勇敢的小熊是谁吗 |
| 情感抓取 | 直接展示强情绪 | 主角哭着说「我讨厌我自己」 |
| 悬念物体 | 展示神秘物品 | 一个发光的盒子谁都不敢打开 |
| 角色反差 | 身份与环境冲突 | 一只小熊在北极卖冰淇淋 |

规则：开场钩子必须同时满足好奇+共情。

### 中间钩子（每3个节点一个小反转）
- 信息反转：揭示新信息改变对之前的理解
- 角色反转：真实意图被揭示
- 目标反转：追求的并不是真正需要的
- 关系反转：敌人变盟友，盟友暂时离开

规则：小反转的目的是推动情感弧线，不是为了反转而反转。

## D. 儿童故事特有的情感叙事要求

### 核心五段式结构（3-8岁儿童）
愿望→障碍→坚持→突破→分享

- 愿望必须是具体的、可见的、可共情的
- 障碍必须是具象的（大怪物/坏天气/迷路）
- 坚持必须展示为「反复尝试」
- 突破必须来自「发现了什么」或「得到了帮助」
- 分享是儿童故事最重要的环节

### 情感可视化对照表
| 内心状态 | 可视化方法 |
|----------|-----------|
| 开心 | 跳跃+笑脸+明亮色彩+音乐音符 |
| 难过 | 低头+流泪+灰色/蓝色调+下雨 |
| 害怕 | 蜷缩+颤抖+深色阴影+大对比 |
| 勇敢 | 抬头+握拳+暖色光芒+向前迈步 |
| 困惑 | 歪头+问号+旋转的线条 |
| 顿悟 | 灯泡亮起+眼睛睁大+微笑 |
| 孤独 | 周围空白+角色变小+冷色调 |
| 温暖 | 拥抱+暖黄光+小动物围绕 |

### 重复中的变化（3-8岁关键机制）
三小一大：同一结构重复3次+第4次变化。
前三次失败的模式可以完全相同，第四次的变化必须明显不同。

### 价值观闭环
每个儿童故事必须有清晰的闭环：
- 「我不够好」→「独一无二就是最好的」
- 「我做不到」→「试试看就知道了」
- 「我没有朋友」→「勇敢说嗨就有了」
- 「我想要那个」→「分享更快乐」
- 「我做错了」→「道歉需要勇气」
- 「我不一样的」→「不一样才特别」

规则：价值观必须通过故事自然展现，不能用旁白说教。

## E. 可嵌入system prompt的压缩版

```
【故事感总纲】
核心原则：你欣赏一个角色的努力拼搏，而不是他本身。故事是角色在克服困难中展现的真实成长轨迹。
叙事结构：开场钩子（3秒内好奇）→情感建置（让观众关心角色）→冲突升级（3波难度递增的阻碍）→转折点（最低谷到新希望）→高潮（情感释放）→情感闭环（开场问题=结尾答案）。
情感曲线：必须是波动上升，不是直线。从好奇→微开心→挫折→回升→大挫折→最低点→希望→高潮→温暖满足。
钩子设计：开场必须反直觉或带疑问，每3个节点一个小反转，结尾留10%想象空间。
儿童故事专属：愿望→障碍→坚持→突破→分享五段式。情感变化用表情/身体/色彩可视化。重复三次+第四次变化。价值观通过故事自然展现不说教。
一句话速记：愿望→阻碍×3→最低谷→成长→突破→温暖闭环，情感波动上升，开场问题=结尾答案。
```
