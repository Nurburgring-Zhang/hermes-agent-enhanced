## 维度库标注规范

### B01_场景环境.txt
```
[indoor_only] 描述...    # 只含室内元素
[outdoor_only] 描述...   # 只含户外元素
[mixed] 描述...          # 同时含室内+户外→丢弃
[碎片] → 长度<10字→删除
```

### A02_发型.txt
```
[NATURAL] 描述...       # 天然发色→亚裔可用
[UNNATURAL] 描述...     # 非天然发色→亚裔禁用
```

### A04_表情眼神.txt
```
[NATURAL] 描述...       # 天然瞳色→亚裔可用
[UNNATURAL] 描述...     # 非天然瞳色→亚裔禁用
```

### D01_服装款式.txt
```
[MALE] 描述...          # 男性专属服装
[FEMALE] 描述...        # 女性专属服装
[UNISEX] 描述...        # 中性服装(男女均可)
```

### D02_配饰鞋帽.txt
```
[MALE] 描述...
[FEMALE] 描述...
[UNISEX] 描述...
```

### A05_姿势.txt
```
[STAND] 描述...
[SIT] 描述...
[LIE] 描述...
[KNEEL] 描述...
[BEND] 描述...

[UNKNOWN] 描述...
```
