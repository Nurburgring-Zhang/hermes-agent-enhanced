# 智影7步SDLC开发流程 (2026-06-10实战固化)

每次开发任务必须按此流程执行,不可跳过。双AI互审贯穿全程。

## 流程

### Step 1: 全网检索
web_search + session_search + fact_store 三路并行
产出: 检索摘要+相关链接+经验教训

### Step 2: 全局观念
读AGENTS.md/SOUL.md/生产规范/相关skill
产出: 规则约束+可复用模式+注意事项

### Step 3: 需求分析
delegate_task子Agent深度分析
产出: 需求文档+功能清单+优先级

### Step 4: 功能设置
架构设计+技术选型+方案评审
产出: 设计文档+选型理由

### Step 5: 软件开发
TDD驱动+真实实现(禁止占位符)
产出: 可运行代码+测试+文档

### Step 6: 审核测试
测试+对抗式验证(双视角)
产出: 测试报告+验证记录

### Step 7: 交付上线
复盘+Memory提取+进化候选
产出: 交付物+复盘报告

## 双AI互审
每步执行前: pre_review() — 高风险拦截+步骤完整性检查
每步执行后: post_review() — 异常检测+连续失败检测
连续3次失败: 建议切换模型或重新规划
