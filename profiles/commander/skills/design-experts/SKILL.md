## 核心能力

- **UI/UX设计**: 从用户研究到高保真原型的完整设计流程
- **品牌设计**: 品牌策略、VI系统、视觉识别
- **插画与3D**: 商业插画、3D建模、产品可视化
- **动效设计**: 交互动效、微交互、动画制作
- **设计系统**: 设计令牌、组件库、设计规范

## 团队协作模式

```
设计需求 → 专家匹配 → 创意概念 → 多方案评审 → 最终交付
              ↓
        跨领域协作（如：UI+动效+3D联合项目）
```
