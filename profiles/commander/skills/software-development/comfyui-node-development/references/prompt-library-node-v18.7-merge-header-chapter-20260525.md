### 3. 短剧总纲：合并核心世界观与通用设定

`drama_header`从一行简短描述改为完整的4板块总纲，放在所有镜头内容之前：
- 【通用基础设定】角色信息
- 【核心世界观之场景通用设定】场景描述
- 【核心世界观之氛围与画质标准】画质/色彩/光线/节奏规则
- 【核心世界观之声音设定】同期声/音效

system prompt中告诉AI："每个【镜头N】直接从镜头脚本内容开始，不再重复输出通用设定与核心世界观（该部分已置于总纲中）"

### 4. 短剧画面描述：800字→200字

system prompt中画面描述字数从800字以上降为200字以上（用户指定）。

### 关键设计规则

1. **总纲和system prompt是分离概念**——总纲（header）是拼接到输出内容前面的定义段，system prompt是发给AI的内部指令。改动时必须分别处理。
2. 当用户提供详细的参考示例时，**复制结构（标题/布局/字段顺序）但不复制内容（角色名/地点/情节）**。
3. 每个分镜脚本的4个字段（景别/台词/运镜/用途标注）独立输出，不融入正文。
4. 总纲中的核心世界观设定放在所有镜头前面，每个镜头直接从分镜脚本开始。
