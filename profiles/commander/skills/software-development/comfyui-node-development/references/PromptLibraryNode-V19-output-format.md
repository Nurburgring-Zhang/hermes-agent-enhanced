# PromptLibraryNode V19 — 模板输出格式与排版约束（2026-05-27补充）

## 输出端口路由

所有22种模式（故事板9种+绘本+短剧+儿童4种+设计7种）的输出全部走 **「模式输出」** 端口。

```python
# Python端分派：
elif 模式选择 in MODE_CATEGORIES_STORYBOARD:
    mode_output = self._process_storyboard_mode(...)  # ❌ 不是final_prompt
elif 模式选择 == "绘本模式":
    mode_output = self._process_picture_book_mode(...)  # ❌ 不是final_prompt
elif 模式选择 == "短剧模式":
    mode_output = self._process_short_drama_mode(...)
elif 模式选择 in MODE_CATEGORIES_CHILD:
    mode_output = self._process_child_mode(...)
elif 模式选择 in MODE_CATEGORIES_DESIGN:
    mode_output = self._process_design_mode(...)
```

RETURN_NAMES 中"模式输出"是第2个输出端口。

**永远不要把模式输出写到final_prompt端口。** final_prompt只用于"关闭"模式（传统提示词库/AI生成）。

## 总纲格式：五维设定

所有模板的总纲必须按以下5个维度展开。**不要用 `## 一、` `## 二、` 这样的编号标题。**

```
# 【模式名总纲】
【角色物品设定】
    角色外貌、体型、服装、关键道具/物品的描述。
    注意：用户输入可能有换行，每行都需要缩进。

【场景设定】
    环境空间、时间光线、氛围特征。整体风格和色调倾向。

【氛围与画质标准】
    视觉风格、叙事方式、画质标准。禁止抽象情绪词。

【声音设定】
    配乐/音效/旁白的设计方向。

【核心叙事方向】
    镜头/页数安排，故事推进逻辑，节奏规划。
---

[具体内容开始]
```

### 排版硬性规则（不遵守会被骂）

1. **缩进用4个空格，不是\t** — `\t`在f-string拼接时可能被转义
2. **用户输入的多行文本要处理换行** — `desc.replace("\n", "\n" + IND)` 确保每行都有缩进
3. **总纲末尾必须有分隔线** — `\n---\n\n` 分隔总纲和具体内容
4. **每个维度之间的空行** — 使用`\n\n`，不要用`\n`
5. **总纲和正文之间不能直接拼接** — 总纲 → `\n---\n\n` → 正文，确保空行分隔
6. **叙事功能不能和时长写在同一行** — 
   ```
   ❌ **时长**：4秒 | **叙事功能**：建立场景
   ✅ **时长**：4秒
   ✅ **叙事功能**：建立场景
   ```

### 五维内容示例（结构参考，内容以用户输入为准）

```
【角色物品设定】
    一只雪白的小兔子，耳朵竖得高高的，粉红色的鼻尖。
    一只圆滚滚的小野猪。
    道具/武器：待补充。

【场景设定】
    清晨的阳光下，一片绿油油的菜园里，泥土湿润而松软。
    整体风格为电影感，色彩基调偏向暖色调。

【氛围与画质标准】
    电影分镜叙事，景别以多种交替为主。镜头语言注重叙事节奏和情绪表达，禁止使用抽象情绪词。

【声音设定】
    根据场景氛围搭配环境音效和配乐。

【核心叙事方向】
    共8个镜头，围绕主题展开。每个镜头用饱满的画面描写，让读者能清晰想象出画面。
```

## Python代码实现模式

```python
IND = "    "
char_desc_clean = (character_desc or "待定角色").replace("\n", "\n" + IND)
env_desc_clean = (env_desc or "待定场景").replace("\n", "\n" + IND)

header = (
    f"# 【{mode}总纲】\n"
    f"【角色物品设定】\n"
    f"{IND}{char_desc_clean}\n\n"
    f"【场景设定】\n"
    f"{IND}{env_desc_clean}\n"
    f"{IND}整体风格为{style}，色彩基调偏向{color_tone}。\n\n"
    f"【氛围与画质标准】\n"
    f"{IND}...\n\n"
    f"【声音设定】\n"
    f"{IND}...\n\n"
    f"【核心叙事方向】\n"
    f"{IND}...\n"
)

# 拼接时必须用分隔线+空行
return f"{header}\n---\n\n{ai_result}"
```

## 内容密度约束（模板级别）

- **总纲每个板块3-5句话**，充分描写但禁止数值参数
- **镜头/画面/页面描述3-6句话**，根据情节调节
- **禁止内容**：数值参数(焦距mm/色温K/分辨率dpi/角度°)、器官名称、肌肉名称、微观生物学细节
- **允许内容**：用视觉化叙事语言写角色动作、场景氛围、光影变化、情绪暗示
