# 2026-06-01 符号清零+变化标注纯净化

## 背景
格林主人在看到AI输出中出现 `**时间空间锚定**` `**变化标注**` 等大量符号后，要求彻底清除所有无用符号。

## 三轮演变

| 轮次 | 规则 | 问题 |
|------|------|------|
| V20.5原始 | 五维度变化标注 + emoji符号 + 无变化写「无」 | emoji出现在输出中 |
| 第一轮 | 五维度中文文字 + 无变化整段不输出 | 仍有「变化标注：」字段 |
| **第三轮** | 只用分镜场景/角色特征字段表示变化，追踪维度缩减为场景/角色两类 |

## 关键文件修改

### 总纲角色物品设定缩进修复
**文件**: `__init__.py` L567-575
**修改**: `char_desc_clean` 处理逻辑加入strip和缩进确保

### 场景设定末尾句号去掉
绘本/短剧/儿童 3个总纲中的 `{env_desc or '待定场景'}。` → `{env_desc or '待定场景'}`

### 变化标注系统彻底重构
删除了所有独立的「变化标注：」字段，改为仅通过`分镜场景`和`角色特征`两个字段承载变化信息。

**修改位置**:
1. `format_templates` 中电影/广告/动画/漫画 — 删除末尾"变化标注："行
2. 故事板铁律5 — 从7行缩为1行
3. 绘本输出格式 — 第4项从"变化标注"改为"分镜场景"
4. 短剧角色设定 — 删除"场景变化标注"段落
5. 短剧铁律4 — 重写
6. 儿童格式一/二/GIF/绘本 — 统一替换

### system prompt符号清除清单
| 符号 | 替换方案 |
|------|----------|
| `# 输出格式（...）` | `输出格式（...）` |
| `# xxx创作原则` | `xxx创作原则` |
| `# 角色设定` | `角色设定` |
| `# 画面铁律` | `画面铁律` |
| `- 列表前缀` | 纯文字描述 |
| `🔴 😊 🎉` | 删除或文字描述 |
| `- 分镜场景：` | `分镜场景：` |
