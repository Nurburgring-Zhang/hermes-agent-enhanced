# PromptLibraryNode V20.5 — 最终版格式化规则（格林主人 2026-05-31）

## 总纲规则（所有22种模板强制）

### 总纲格式（统一）
【XX总纲】
整体视觉风格：
    场景描述内容
氛围与画质标准：
    画质/氛围描述
核心叙事设定：
    情节/镜头数/叙事方向
角色设定：
    角色1：外貌/服装/标志物
    角色2：外貌/服装/标志物
环境设定：
    场景：地点
    时间：时间段
    氛围：氛围描述
声音设定：
    音效/配乐/配音风格

### 总纲铁律
- **无 `#` 前缀**：总纲标题直接以 `【】` 开头
- **无 `【】` 包裹key**：section标题不用【】 —— 用"整体视觉风格："、"氛围与画质标准："、等中文冒号格式
- **无 `---` 分隔**：总纲与正文直接空行衔接，不用分隔线
- **无空行**：总纲内各section之间不空行
- **角色设定和环境设定必须在总纲内**，不由AI在正文开头单独输出【角色设定】【环境设定】

## 分镜输出规则

### 叙事功能
- **仅在必要时输出**，不是每个分镜/页面都必须有
- 当需要说明镜头在故事中的作用时才写，一句话简写

### 分镜场景 / 角色特征（重要）
- 这两个字段**仅在发生变化时才输出**
- **场景改变时**：输出 `分镜场景：新环境描述`
- **角色特征改变时**：输出 `角色特征：变化后的角色描述`
- **两者都改变时**：同时输出两个字段
- 不变化时不写，不罗列无效维度

## 符号规则
- **禁止所有 `**` 加粗符号**
- **禁止所有 `- **` 组合**
- **禁止所有 `- ` 列表符号**（在模板内容中）
- 使用纯文本描述

## 模板差异性
- 每个模板的规则必须**符合模板自身类型**
- 场景切换规则按模板定制（电影用远景、广告用产品锚点、漫画用大格等）
- 不能所有模板共享同一套机械描述

## 改前备份铁律
- **任何修改/删除之前必须先备份**
- 备份到 `/mnt/d/Hermes/备份/` 目录
- 含单文件修改、sed批量替换、crontab变更等
