# 变化标注规则演化史（格林主人指令轨迹）

## 版本迭代

### V1 — 旧版（V20.4及之前）
- 故事板：两维度（场景变化时写分镜场景，角色特征变化时写角色特征）
- 绘本：两维度（场景变化+角色特征变化）
- 儿童GIF：三维（场景/角色/道具）
- 短剧：两维度（场景变化+角色特征变化）

### V2 — 五维度emoji版（V20.5，2026-05-29）
- 统一升级为五维度：场景/角色/服装/道具/时间
- 使用emoji标记：🔄👤👗📦☀️
- 规则：变化几维写几维，无变化写「无」
- 覆盖：故事板9种+绘本+短剧+儿童4种

### V3 — 总纲编号格式（2026-05-31 第一轮修改）
- 总纲从分段式改为编号1~7格式
- 分镜场景/角色特征移至画面之前（新顺序：景别→分镜场景→角色特征→画面）
- 景别标注「（始终输出）」

### V4 — 最终版（2026-05-31 第二轮修改，当前）
**触发：** 格林主人展示了实际输出并指出：
1. 总纲内不要有空行
2. 无变化时整个变化标注不输出（不要写"变化标注：无"）
3. 只写有变化的维度（不要写无变化=无）
4. 维度名称用中文文字，不用emoji/符号

**最终规则：**
- 变化标注仅在场景/角色/服装/道具/时间有变化时输出，无变化时整段不出现
- 只写有变化的维度名称，不写无变化的维度
- 维度名称用中文文字：场景变化/角色变化/服装变化/道具变化/时间变化
- 禁止使用emoji（🔄👤👗📦☀️）或特殊符号
- 分镜场景和角色特征字段同样：仅在变化时输出整行，无变化时行消失
