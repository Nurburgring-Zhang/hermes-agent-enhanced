# Prompt Library Node — 大规模提示词生产经验

## 核心策略

### 方法1: 模板组合脚本（最快，每秒500条）
使用 `massive_generator.py` — 组合式生成引擎，从2000风格×2000穿搭×2000姿势库随机组合。

适用场景：用户要求"快速产出10万条"，质量可接受模板组合。

### 方法2: AI生成+模板库（每分钟9条，质量高）
使用cronjob并行生成，每5分钟9条。

适用场景：每条需要AI级别的独特性和创造力。

## 去重策略

- 跨文件去重：用前20字+方向前缀保证区分
- 文件内去重：前20字hash
- 过短的提示词（<200字）直接不写入

## 后期处理

### 光影描述清理（用户要求）
删除"泛着"→"带着", "光晕"→"光", "镀上"→"染上", "红润"→"透亮", "倾泻"→"照入"
清理量：约30万处修改

### 面部色彩清理（用户要求）
删除腮红/晒红/晒伤/红润/潮红/绯红/嫣红/酡红/晕红等
清理量：27,800处
