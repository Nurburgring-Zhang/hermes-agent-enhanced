# modern_art_prompts_02.txt 专用格式规范

> 适用于 `D:\Hermes\1000000提示词\10万提示词\modern_art_prompts_02.txt` 及同系列文件（社交网红照类 prompt 库）

## 文件格式

- 纯文本文件，每**两个相邻prompt之间空一行**
- 每个prompt以 `  编号|` 前缀开头（空格+数字+竖线）
- 但**追加时不需要写编号**（后续统一格式化时加）
- 每行一条prompt，prompt本身是**单段文字**，**不要硬换行**
- 已有文件中每个prompt都符合：**580-730字符**（中文字符，用 `len()` 测量）

## 九层模板结构（本文件专用变体）

与SKILL.md中的7维度框架不同，本文件使用9层顺序结构，每层有固定字符占比：

| 层 | 内容 | 字符占比 | 示例关键词 |
|----|------|---------|-----------|
| 1 | **光线/时间/空间氛围** | ~18% | "清晨六点半的微光穿过亚麻窗帘..."/"雨后光线从落地竹帘缝隙筛进来..." |
| 2 | **镜头设定** (焦距+角度+构图) | ~6% | "镜头用85mm定焦从侧面平视拍摄，构图采用中轴对称但略微打破..." |
| 3 | **主体姿态/动作** | ~15% | "她侧身蜷坐在藤椅上，一条腿蜷起踩在坐面上..." |
| 4 | **头发细节** | ~8% | "一头乌黑亮泽长直发披散肩头，发质顺滑如绸缎..." |
| 5 | **皮肤/面部/妆容** | ~8% | "皮肤是温润的米白色在侧光中呈现柔和光泽..." |
| 6 | **服装穿搭** | ~12% | "身上穿着奶油白色粗针织开衫没有系扣，内搭米白棉质T恤..." |
| 7 | **前中后景** | ~12% | "前景虚化的藤编椅背纹理...中景她放松倚靠的轮廓...后景窗外..." |
| 8 | **配色系统** (传统色名四色交织) | ~10% | "通篇配色以月白为基底，叠加油黄、淡缃、烟霞红四色交织..." |
| 9 | **光源/光效 + 出圈要素** | ~11% | "光从东窗斜射进来...出圈要素是笔尖与宣纸间那两寸悬而未落的真空距离" |

## 必须遵守的规则

### 1. 字符数铁律
- **每条prompt严格580-730中文字符**（用 python3 `len(text)` 验证）
- 超出730：删除冗余修饰词、合并并列短句
- 低于580：增加材质细节（头发纹理、皮肤质感、服装褶皱）

### 2. 开头格式
- 直接以**光线/时间背景描述**开头（1-2句），不要"一幅画面中"/"展示的是"/"这是一幅"
- 示例正确开头："清晨六点半的微光穿过亚麻窗帘的缝隙..."
- 示例错误开头："一幅法式慵懒风格的画面中，一位女子..."

### 3. 标配要素（每一条都必须有）
- ✅ **焦距**（35mm/50mm/85mm中的一种）
- ✅ **传统色名四色交织配色**（结尾处写一组四个传统色名及其分布）
- ✅ **出圈要素**（最后1-2句的视觉钩子，如物象错位/超现实悖论/风格致敬）
- ✅ **前景-中景-后景**分层描述
- ✅ **光源方向+阴影描述**

### 4. 风格覆盖（每条选1-2种混合）
- **法式慵懒风**：海魂衫/法式单椅/亚麻西装/巴洛克细节/香槟色/羊角面包/咖啡
- **韩系奶油风**：粗针织开衫/阔腿裤/奶油白/冰美式/水光肌/毛绒拖鞋/模糊柔光
- **新中式风**：改良交领/苎麻/墨砚/老榆木案/竹帘/宣纸/低发髻/素银簪
- **法式冷感风**：烟灰大衣/真丝吊带/狼尾发/马丁靴/画室/北向天窗/冷白皮
- **森系自然风**：藤编/棉麻/赤足/花园/晨光/洗衣床单/植物
- **复古美式风**：motel/匡威/牛仔/威士忌/百叶窗光带/褪色面料

### 5. 镜头词汇（固定用这些）
- **镜头**：35mm定焦 / 50mm定焦 / 85mm定焦 / 28mm广角
- **拍摄角度**：平视 / 略高俯拍 / 低角度仰拍 / 侧面平视
- **构图**：人物居三分线左/右侧 / 人物居中画面中央偏左/右 / 中轴对称略微打破
- **景别**：中景 / 中近景 / 全景（整套提示词统一用中景为主）

### 6. 传统配色模板（每条约用一组）

**常用四色组合**：
- 月白 + 油黄 + 淡缃 + 烟霞红 — 清晨/新中式场景
- 退红 + 青灰 + 老蜜色 + 竹青 — 慵懒午睡场景
- 酒红 + 普鲁士蓝 + 琥珀 + 蜜金 — 黄昏/都市场景
- 奶油白 + 暖棕 + 浅灰 + 淡粉 — 韩系奶油场景
- 藕荷 + 月白 + 鸦青 + 竹青 — 新中式/宋韵场景
- 群青 + 赭石 + 烟灰 + 钴蓝 — 画室/创作场景
- 鼠灰 + 月白 + 青黑 + 竹青 — 书房/抄经场景
- 暮紫 + 暖金 + 靛蓝 + 米白 — 城市暮色场景

### 7. 出圈要素类型
- **物象错位**：围裙颜料斑驳既记录创作史又像随机抽象画
- **超现实悖论**：窗外天际对角线倾斜暗示空间失重
- **反差缺陷**：在完美画面中故意留下咖啡杯口干涸痕迹
- **风格致敬**：致敬南宋院体画风，将姿态转化为水墨工笔线条
- **时间叙事**：杯口干涸水渍与杯内未饮尽液面之间形成的时间流逝

## 生成验证流程

```bash
# 1. 生成内容写入临时文件
cat > /tmp/test_prompt.txt << 'EOF'
你的prompt内容...
EOF

# 2. 验证字符数
python3 -c "print(len(open('/tmp/test_prompt.txt','r',encoding='utf-8').read()))"

# 3. 确认在580-730范围内后追加
cat /tmp/test_prompt.txt >> /path/to/modern_art_prompts_02.txt

# 4. 追加后统计
python3 -c "
import re
text = open('/path/to/modern_art_prompts_02.txt','r',encoding='utf-8').read()
prompts = [p.strip() for p in re.split(r'\n\s*\n', text.strip()) if p.strip()]
print(f'Total: {len(prompts)} prompts')
for p in prompts[-5:]:
    print(f'  {len(p)} chars')
"
```
