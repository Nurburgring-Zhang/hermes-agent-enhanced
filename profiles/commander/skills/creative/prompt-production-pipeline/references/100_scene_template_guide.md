# 100-Scene Template Guide

## Structure Overview
10 categories × 10 templates = 100 scene templates for massive prompt generation.

Each template: Scene + Character + Clothing + Accessories + Activity + Props + Materials + Lighting + Color + Composition + Dynamics + Atmosphere + Breakout element.

## When to Use
- Generating 10K+ prompts from dimension libraries
- Need systematic coverage of diverse visual scenarios
- Ensuring no two prompts are too similar in structure/themes

## Dimension Counts
| Count | Instances | Notes |
|-------|-----------|-------|
| 12-14 | 2-3/100 | Simple scenes |
| 15-17 | ~50/100 | Main sweet spot |
| 18-20 | ~40/100 | Rich, professional |
| 20+ | ~5/100 | Maximum detail |

## 10 Categories

### Group 1: Daily Life & Lifestyle (#01-10)
- Morning bed laziness, afternoon tea, bathroom wet hair, bed selfie, balcony coffee, dressing table makeup, window reading, headphones music, yoga stretch, deep night alone
- **Keywords**: 居家氛围感, 慵懒, 生活写真, 私房
- **Lighting**: window light, warm tungsten, morning sun
- **Vibe**: cozy, intimate, lazy, warm

### Group 2: Urban Street & Fashion (#11-20)

- **Lighting**: direct sun, neon, flash + studio, street lamp

- **Keywords**: 故事感, 电影级, 情绪, 叙事

- **Vibe**: moving, powerful, nostalgic, hopeful

## Avoiding Duplication
- Each template within a group uses different lighting (always vary: front/side/back/top)
- Vary scene types across groups: natural → urban → indoor → fantasy
- Vary emotional tone: happy → sad → sexy → mysterious → hopeful
- Vary color palettes dramatically between adjacent templates

## Quality Checklist per Template
- [ ] ≥12 dimensions (target 16-18)
- [ ] Natural prose paragraph, no numbered lists or headers
- [ ] No比喻词 (仿佛/犹如/就像/好似/宛如/如同)
- [ ] No three-hand violations
- [ ] Scene and lighting are consistent
- [ ] Weather and clothing are consistent
- [ ] Has emotional/atmospheric dimension
- [ ] Has color palette description
- [ ] Has compositional/photo technique reference
- [ ] Breakout/cross-platform appeal element present
