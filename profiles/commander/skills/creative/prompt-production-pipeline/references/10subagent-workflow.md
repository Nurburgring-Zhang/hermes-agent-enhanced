# 10万条子代理并行生产工作流（2026-05-25）

## 背景

格林主人要求深度分析 D:\Hermes\1000000提示词\高质量模板 内的文档后，启动10个子代理，每个生成10,000条高质量prompt，最终汇总10万条。

## 前置分析步骤

### Step 1: 文档深度扫描

扫描文件夹 D:\Hermes\1000000提示词\高质量模板\ 内的所有文档，提取：

1. **现有维度库清单**：17+个维度库（A01-D06），每条100-1000字片段
2. **风格库内容**：15_美学风格库.txt（500条风格描述）、100个场景模板
3. **破圈逻辑**：风格致敬（宫崎骏/穆夏）、超现实悖论（巨人传送门）、反差叙事（MS Paint涂鸦）
4. **组合规则**：12必含维度+3可选维度、禁止三手/裸体+服装/场景矛盾/比喻词

### Step 2: 联网检索（可选补充）

检索现代艺术展/美术馆/Artstation/AIGC站最受欢迎的视觉艺术作品风格特征，补充：
- 1000种可自由混搭的现代视觉艺术风格
- 在美术馆+艺术站+社交平台交汇地带受欢迎的作品特征
- 破圈逻辑的具体案例

### Step 3: 风格-场景-配色矩阵构建

构建10×10×10矩阵：
- 10个风格大方向（每个子代理1个）
- 每个风格大方向拆解10个子风格
- 每个子风格匹配10种场景
- 每个组合匹配3种配色方案 + 3种情感基调

## 子代理系统提示模板

完整提示模板（复制到delegate_task中）：

```
你是Hermes的子代理S[N]（共10个子代理之一）。

【你的专属领域】
风格方向：XXX（如古典油画与现代主义）
场景主题：XXX（如贵族花园、舞会）
情感基调：XXX（如优雅、惆怅、静谧）
目标数量：1000条
输出文件：D:\Hermes\1000000提示词\10万提示词\modern_art_prompts_S[N].txt

【质量铁律】
1. 每条500-800字中文，完整段落，不要标题/序号/分段
2. 开篇必须点明风格名称+风格特征细节描述

【执行方式】
你必须逐批生产，每批生成20条后写入文件，再继续下一批。
```

## 质量检查流程

生成完成后做全局质量检查：

1. **全局去重**：使用 `textdistance.jaccard_similarity` 跨文件去重
2. **三手检测**：正则检测 `手` 出现超过2次的条目
3. **比喻词检测**：检测 `仿佛/犹如/就像/宛若/如同/宛如`
4. **场景矛盾检测**：检测室内+户外关键词混用
5. **字数检测**：低于450字或超过850字的标记
6. **风格开篇检测**：前20字是否包含风格关键词
