# 信达雅翻译流程

## 何时使用
当需要将英文长文本（论文、技术文档、书籍章节、商业报告）翻译为高质量中文时使用。不适合短句/对话翻译。

## 三层翻译流程

### 第一层：信（Faithfulness）—— 逐句忠实
- 每句英文都对应准确译文
- 数字、逻辑关系、专有名词完全保留
- 术语翻译统一（全文保持一致）
- 不自行添加原文没有的含义

### 第二层：达（Expressiveness）—— 中文可读
- 按中文"话题-评论"结构重排语序
- 拆分过长的英文复合句（单句不超过50字）
- 转换被动句为主动句
- 英文的"it is...that"结构转为中文自然表达

### 第三层：雅（Elegance）—— 消除翻译腔
- 检测并消除"的的的"连续（不要超过2个"的"字连用）
- 消除"之一"的滥用
- 消除名词化过度（"实施该方案的过程"→"实施这个方案"）
- 检查被动句比例（中文被动句不超过20%）

## 风格控制
| 风格 | 适用场景 | 特点 |
|------|---------|------|
| 学术严谨 | 论文、报告 | 术语精确，句式完整，保留所有引用 |
| 技术实用 | 技术文档、手册 | 简洁明了，步骤清晰，专业术语保留英文原名 |
| 商业流畅 | 商业报告、方案 | 语言流畅，有说服力，适合中文读者 |
| 文学优雅 | 书籍、散文 | 语言优美，有文学性，可适当意译 |
| 通俗易懂 | 科普、大众内容 | 深入浅出，去除学术术语，用类比解释 |

## 输出格式
1. 原文段落逐段对照
2. 术语表（英中对齐）
3. 翻译说明（如果有意译/特殊处理之处）
4. 最终中文版本（完整、可读、无翻译腔）
