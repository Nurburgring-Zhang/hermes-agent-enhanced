# Autonomous AI Agents (自主 AI 代理)

此分类包含自主 AI 代理的生成和编排工具，可启动独立的 AI 编码代理进程、委派子任务并协调并行工作流。

## 触发条件
- 用户提及此功能相关关键词时
- 遇到此领域的标准问题时
- 需要执行该领域的标准操作时


### claude-code
Claude Code 代理集成，启动和管理 Claude 编码代理会话。

### codex
Codex 代理集成，使用 OpenAI Codex 进行代码生成和任务执行。

### hermes-agent
Hermes Agent 自身的管理工具，配置和操控 Hermes 代理的行为。

### opencode
OpenCode 代理集成，启动和管理开源的编码代理工作流。
