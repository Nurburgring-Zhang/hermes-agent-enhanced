## 关键经验

1. **API扩写失控是主要问题源**: 维度组合阶段防得住85%的问题，APl扩写阶段会自己编造矛盾内容
2. **三类最致命API失控**: 
   - 光照系统混合(自然光+舞台光+夜景)
   - 人物数量自己增加(一名→三人)
   - 摄影师术语混入(对焦/光圈/景深)
3. **后处理规则必须覆盖语义级矛盾**: 不能只靠关键词正则，还需要qc_two用逻辑判断
4. **gen_one()前置过滤最关键**: MALE_OCCUP检测"士兵"让D01过滤女性装束，从源头防
5. **试产10条必须全量分析**: 不能只看1-2条合格就觉得"可以了"
