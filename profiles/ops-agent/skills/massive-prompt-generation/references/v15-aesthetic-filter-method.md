在v14（直出原始语料）基础上增加了200+个审美/艺术/时尚/潮流关键词过滤，确保每条输出都是高审美内容。

## 速度

- 加载语料: ~1s

- 批量100万条（20文件×5万）: ~13s

## 去重机制

用前100字作为key，去重范围包括：
1. 本次批次内的条目
2. 之前所有v14/v15批次
3. 之前所有百万批次

## 关键决策

- 保留成人内容（不过滤性相关词）
- 最大字数上限1500（部分详细描述需要）
- 每文件50000条是为了文件大小可控（约15-20MB/文件）
