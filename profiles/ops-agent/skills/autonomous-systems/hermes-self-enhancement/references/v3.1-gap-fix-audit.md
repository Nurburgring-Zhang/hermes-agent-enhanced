## 发现的3个严重Bug(修复)
1. `sqlite3.Connection.lastrowid` → `cursor.lastrowid` (Python 3.12变化)
2. Python name mangling: `__init_core_v2()` → `init_core_v2()` (双下划线触发Python类名修饰)
3. Merkle树`add_step`存储hex字符串而非digest字节(树节点hash计算错误,导致验证失败)
