## 关键教训

1. **过滤误杀是最大风险** — 单字过滤词("若""胸""指")会产生大量假阳性
2. **Python隐式字符串拼接是隐蔽语法错误** — `"床上" "沙发"` → "床上沙发"
3. **注释承诺不实现 = 功能缺失** — "也要检测'和'连接不同性别"注释存在但代码不存在
4. **三手检测应该是逻辑矛盾检测，不是计数检测**
5. **场景矛盾的根因在维度库数据，不在组合引擎逻辑**
