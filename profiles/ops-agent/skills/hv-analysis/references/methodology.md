# hv-analysis 方法论详述

## 方法论起源

横纵分析法（Horizontal-Vertical Analysis）由**数字生命卡兹克**（GitHub: KKKKhazik）提出并开源。

原始仓库：https://github.com/KKKKhazik/khazix-skills

## 核心思想

> **"做深度分析，不能只看一个维度。"**

横纵分析法的核心假设是：任何值得分析的议题，都至少需要从三个维度解构——时间（纵向）、空间（横向）、以及竞争力量（战略叠加）。

## 与其他分析框架的关系

### 继承关系
- 纵向分析 ← 历史分析法 / 时间线分析法
- 横向分析 ← 竞争分析 / 对标分析 / 产业链分析
- 战略叠加 ← 波特五力 / SWOT / 麦肯锡三层面法

### 独特性
1. **简化不简化逻辑**：只用3个维度，但每个维度都要求深度而不是广度
2. **结构化输出**：固定输出格式，便于对比和归档
3. **可操作性强**：最后一章必须有"行动建议"和"风险提示"，避免只分析不解决问题

## 适用场景

| 场景 | 适合 | 不适合 |
|------|------|--------|
| 行业趋势分析 | ✅ 有清晰时间线和玩家 | ❌ 全新领域（无历史数据） |
| 公司战略研究 | ✅ 公开信息充分 | ❌ 黑盒公司（信息太少） |
| 技术路线预判 | ✅ 技术演进有规律 | ❌ 纯投机炒作（无基本面） |
| 用户群体研究 | ✅ 用户分层明显 | ❌ 极小众市场（样本不足） |
| 政策影响评估 | ✅ 有政策时间线 | ❌ 频繁变动的短期政策 |

## 分析层级

根据分析深度，分为三个层级：

### L1: 快速扫描（15分钟）
- 纵向：草拟时间线，标注关键事件
- 横向：列出前5个玩家
- 战略：SWOT列表
- 输出：500-1000字简报

### L2: 标准分析（1小时）
- 纵向：3年时间线 + 转折点识别 + 趋势判断
- 横向：玩家地图 + 产业链 + 标杆案例
- 战略：波特五力 + SWOT + 定位
- 输出：2000-4000字报告

### L3: 深度研究（4小时+）
- 纵向：5年时间线 + 多情景推演 + 量化预测
- 横向：全产业链利润池分析 + 全球区域对比 + 详尽的用户分层
- 战略：完整的波特五力量化评分 + 四种定位备选方案
- 输出：8000-15000字深度报告

## 常见认知偏差与应对

| 偏差 | 表现 | 应对策略 |
|------|------|---------|
| 确认偏误 | 只找证实自己观点的证据 | 强制搜索反方观点 |
| 幸存者偏差 | 只关注成功案例 | 专门分析失败案例 |
| 后见之明 | 事后觉得一切显而易见 | 记录分析时的不确定性 |
| 锚定效应 | 过度依赖第一个信息 | 多源交叉验证 |
| 框架效应 | 被问题的表述方式影响 | 同一问题换多种表述 |
| 近期偏差 | 高估近期事件的重要性 | 严格按时间线排列 |

## 数据源优先级

1. 一手数据（财报、官方数据、专利） > 二手分析 > 媒体报道
2. 近3年数据时效系数: 1.0, 3-5年: 0.7, 5年以上: 0.4
3. 信源权重: 政府/行业协会(0.3) > 上市公司报告(0.25) > 学术论文(0.2) > 头部媒体(0.15) > 自媒体(0.1)

## 评分标准（用于评估分析报告质量）

| 维度 | 优秀(90+) | 良好(75-89) | 及格(60-74) | 不及格(<60) |
|------|-----------|-------------|-------------|-------------|
| 分析深度 | 识别了深度学习因果链 | 有清晰的时间线+转折点 | 有基本的时间线 | 只有碎片信息 |
| 横向广度 | 全产业链+全球对比 | 主要玩家+产业链 | 有玩家列表 | 只有少量信息 |
| 可操作性 | 具体的行动建议+风险量化 | 有明确建议 | 建议较模糊 | 无建议 |
| 数据支撑 | 多源交叉验证+量化数据 | 有数据支撑 | 少量数据 | 纯主观判断 |
| 独特洞察 | 提供了反共识观点 | 有新颖视角 | 常识复述 | 老生常谈 |
| 时效性 | 包含最新事件 | 最近1年数据 | 最近2年数据 | 数据过时 |
