# 2026-06-01 Final Revision: Symbol-Free + Change Tracking Simplification

## 总纲格式（所有模式统一）

```
【{mode}总纲】
1、整体视觉风格：
整体风格为{style}，色彩基调偏向{color_tone}。
2、角色物品设定：
{character_desc}
3、道具/武器：
待补充。
4、场景设定：
{env_desc}
5、氛围与画质标准：
...内容...
6、声音设定：
...内容...
7、核心叙事设定：
...内容...
```

**规则：** 无缩进（内容顶头写），无空行，角色物品设定多角色时直接换行不加句号

```
角色特征：（有大的角色特征变化时输出完整角色描述2-3句，无变化时整行不出现）
画面：...（始终输出）
运镜：
转场：
时长：
叙事功能：（必要时输出）
```

## 儿童视频格式一输出格式

```
时间·空间锚定：清晨·菜园
场景描述：...（始终输出）
动态描述【动态】：...（始终输出）
分镜场景：（有大的场景变化时输出，无变化时不出现）
角色特征：（有大的角色特征变化时输出，无变化时不出现）
旁白/对话：...（始终输出）
特效/TIPS：...（始终输出）
```

## 变化标注的核心原则

1. **只追踪场景和角色变化** — 不再追踪服装/道具/时间
2. **变化时输出完整描述** — 不是简短标注，是类似总纲中的详细描述
3. **无变化时整行不出现** — 字段标题本身都消失
4. **所有其他字段正常输出** — 不受变化规则影响

## system prompt符号禁令

已从所有system prompt中移除：
- `#`（标题标记）→ 纯文字标题
- `##`（二级标题）→ 纯文字
- `**`（加粗）→ 纯文字
- `- `（列表前缀）→ 纯文字
- `🔴`（红色圆点）→ 纯文字
- `😊` `😟` `🎉`（表情符号）→ 纯文字描述
- `🔄` `👤` `👗` `📦` `☀️`（维度符号）→ 中文文字
