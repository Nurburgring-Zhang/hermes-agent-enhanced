# PromptLibraryNode V18.8 — 儿童专用内容生成模块

## Overview

Added a 6th output port (`儿童提示词`) with 4 content modes, age-aware system prompts, 5 art styles, and full structural independence from all other ports.

## Key Design Decisions

### 4 Content Modes (exact output structure per mode)

| Mode | Header | Content Structure | Reference File |
|------|--------|-----------------|----------------|
| 儿童视频格式一 | 【片段N】 | 场景描述 → 【动态】→ 【旁白/对话】→ 【特效/TIPS】 | "糖豆和皮皮" style |
| 儿童视频格式二 | 第X幕 | 角色介绍(仅第一幕) → 【场景】→ 画面描述 → 对话 → 旁白 → TIPS | "蛇下雨天会干嘛" style |
| 儿童微动视频/GIF格式 | 第X页 | 【旁白】→ 【画面】→ 【对话】→ 【动效描述(循环方式)】 | "草船借箭" style |
| 儿童绘本格式 | 第X页 | 画面描述 → 文字(1-2行) | "鲸鱼科普" style |

### Age Group Adaptations

Three age tiers injected into system prompts:
- **0-3岁低幼**: 5-15字/段, 单一动作, 认知主题, 高度重复句式
- **3-6岁幼儿**: 10-30字/段, 简单情节(开头-冲突-解决), 童话/友情/习惯主题
- **6-9岁学龄**: 20-50字/段, 完整起承转合, 勇气/成长/科学主题

### 5 Art Styles

| Style | Description |
|-------|-------------|
| 水彩插画 | 水彩晕染, 色彩柔和通透, 边缘自然过渡 |
| 卡通动画 | 粗轮廓线, 纯色填充, 表情夸张可爱 |
| 彩铅手绘 | 铅笔纹理, 色彩层叠柔和, 温暖质朴 |
| 黏土定格 | 立体感强, 材质真实可见, 色彩饱和温暖 |
| 扁平矢量 | 简洁几何形状, 纯色块面, 现代清新 |

### Implementation Pattern

Each mode has its own `_build_child_*_prompt()` method. A central dispatcher `_build_child_system_prompt(mode, style, age_group)` routes to the correct method by mode name.

Total 6 new methods added:
- `_build_child_system_prompt` — dispatcher
- `_build_child_video1_prompt` — format 1 system prompt
- `_build_child_video2_prompt` — format 2 system prompt
- `_build_child_gif_prompt` — GIF format system prompt
- `_build_child_book_prompt` — picture book format system prompt
- `_get_age_range_for_mode` — duration helper (hardcoded "1-3" for now)

### Structural Rules

1. **Never modify existing ports** — the child module appends after all other logic, just before return
2. **Only fire when** `输出儿童内容 and API地址` — silent skip otherwise
3. **Use same _call_ai pattern** as 故事板/绘本/短剧 — `raw → header + raw → port`
4. **Header format**: `【儿童内容总定义】生成{模式}，主题：{主题}，共{N}个片段/页，画风：{画风}，年龄段：{年龄段}。`
5. **Return is 6-tuple**: add `child_prompt` as 6th element. `_error_result` also returns 6 elements.
6. **All system prompts start with authoritative persona** (世界顶级的儿童xxx专家)

### Safety Considerations

- All content is generated by LLM with alignment — no separate safety filter on the prompt level
- Safety filtering is expected to happen downstream (image/video generation stage), not in the prompt node
- Content is designed for 0-9 year olds — themes, vocabulary, and conflicts are child-appropriate
- The module generates **text prompts/scripts only**, not final media
