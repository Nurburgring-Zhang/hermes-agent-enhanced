## 通道间协同流程

```
用户查询
  |
  v
IntegratedArbiter.search()
  +-- SemanticChannel.retrieve()  [语义相似度, top-20]
  +-- KeywordChannel.retrieve()   [FTS5 BM25, top-20]
  +-- TimelineChannel.retrieve()  [recency排序, top-20]
  +-- SpreadingActivationChannel.retrieve() [多跳扩散, top-20]
  +-- EntityGraphChannel.retrieve() [图谱遍历, top-20]
  +-- HopfieldChannel.retrieve()  [模式补全, top-20]
  |
  v
_signal_filter() [15%阈值, 信号+0.2/噪音-0.3]
  |
  v
增强加权 [语义*1.5, 关键词*1.3, 扩散*1.2, 图谱*1.1]
  |
  v
MD5去重 → top-20排序
```

## 漂移检测参数(从实战调优)

| 参数 | 值 | 说明 |
|------|-----|------|
| 中文n-gram长度 | 2,3,4字滑动窗口 | 覆盖中文所有常见词组 |
| 英文min_len | 3 | 忽略短缩写 |
| OK阈值 | <0.25 | 无漂移 |
| MILD阈值 | 0.25-0.5 | 轻度 -> 引导提示 |
| MODERATE阈值 | 0.5-0.7 | 中度 -> 回退3-5步 |
| SEVERE阈值 | >=0.7 | 重度 -> 重置+复盘 |
| 语义权重 | 0.4 | 目标-上下文距离 |
| 进度权重 | 0.3 | 预期vs实际进度偏差 |
| 完成奖励 | *0.7 | 进度>80%时降低漂移警告 |
