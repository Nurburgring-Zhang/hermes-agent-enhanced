# PromptLibraryNode 总纲格式与分镜变化规则（2026-06-01 更新 v4）

## 总纲格式（所有模板统一 — 编号结构）

**2026-06-01重要更新**：总纲从旧的分段section格式改为编号列表格式。

```text
【xxx总纲】
1、整体视觉风格：
    风格/画风/色调信息。缩进4个空格。
2、角色物品设定：
    角色描述（外貌、体型、服装、标志特征）
3、道具/武器：
    待补充。（AI在内容中自行扩展）
4、场景设定：
    场景/环境描述（空间、时间、光线、氛围）
5、氛围与画质标准：
    画质/叙事标准/景别/运镜
6、声音设定：
    音效/配乐描述
7、核心叙事设定：
    篇幅/结构/节奏/故事方向
```

### 格式规则
- 使用 `N、` 编号前缀（`1、` `2、` `3、`等）
- 每段内容缩进4个空格
- 总纲内禁止空行
- 角色描述在「角色物品设定」section内，不单独输出【角色设定】引导
- 道具/武器统一为「待补充」

## 分镜变化规则

**分镜场景**字段 — 仅在场景发生改变时才在分镜/页面内增加一行，不变化不写
**角色特征**字段 — 仅在角色特征发生改变时才增加，不变化不写
两者都变化时同时增加
**叙事功能** — 仅在需要说明镜头作用时才输出，非必须

## 五维度变化标注（2026-06-01新增）

当场景/角色/服装/道具/时间中任何一个发生变化时，必须在变化发生的首个镜头的画面描述中显式说明：

- 🔄 场景变化：从哪到哪？
- 👤 角色变化：新角色出现/角色离开/角色状态改变？
- 👗 服装/配饰变化：服装/配饰改变？
- 📦 道具/物品变化：关键道具状态变化？
- ☀️ 时间/光线变化：时间推进/光线变化？

变化几维写几维，无变化写「无」。

## 覆盖范围

- 故事板模式（9种）：铁律5已统一升级
- 绘本模式：输出格式维度4 + 创作原则已升级
- 短剧模式：角色设定 + 铁律4已升级
- 儿童内容（4种）：格式一/格式二/GIF/绘本格式均已升级
- 专业设计（8种）：静态设计图，不适用变化追踪

## 常见Bug

1. `get_prompt(self, kwargs)` 应为 `get_prompt(self, **kwargs)` — ComfyUI将所有input参数作为命名关键字参数传入，不修复则报错
2. 批量字符串替换时要注意转义层级
3. 总纲替换用`find("f\"【{mode}总纲】")`会匹配到故事板和儿童两个总纲——必须用后续行区分
