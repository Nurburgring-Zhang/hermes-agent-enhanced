# modern_art_prompts_06.txt 专用格式规范 — 都市夜景/深夜治愈/美食

## 文件定位

`06.txt` 对应「都市夜景/深夜治愈/美食」主题，每条描述一个中国城市的深夜美食摊/店/档口场景，一条完整的沉浸式城市深夜叙事。

## 模板结构：九层叙事法（区别于02系列的7维度结构）

| # | 层 | 内容要点 | 字数占比 |
|---|----|----------|---------|
| 1 | **场景设定** | 一条[城市地名][具体小巷/街区+美食摊位/店铺]场景，交代时间（凌晨1-3点）、天气（雨/雾/晴/海风）、街道氛围 | 8% |
| 2 | **主体穿搭** | 年轻女性，全传统色名+面料材质+单品细节。必须包含：上衣(色+面料+剪裁) + 外搭(色+材质+穿着状态) + 下装(色+面料+长度) + 鞋 + 发饰/发型 + 当前动作(等餐/吃东西/看手机) | 20% |
| 3 | **传统色彩环境调色板** | 以「传统色彩以XX为基调——」引出，列举6-8个环境色块：夜空色、建筑材质色、摊位/设备色、食物色、人物衣物色在环境光下的色偏。使用冒号分隔每个色块点 | 18% |
| 4 | **摄影风格** | 采用[XX相机型号/风格]，主光源来自[具体光源] + [第二光源]的混合，光线对人物面部和环境的渲染效果 | 8% |
| 5 | **光线细节** | 光源方向、色温对比（冷暖对峙）、特殊光学效果（蒸汽透光/霓虹倒影/逆光头发轮廓/水面光斑） | 8% |
| 6 | **构图** | 构图类型（平视/俯拍/远景/特写），人物在画面中的位置，前中后景层次关系 | 8% |
| 7 | **嗅觉/听觉/触觉** | 3-5种气味（食物的+环境的）、3-5种声音（近处的+远处的）、温度/触感（烫/暖/凉/冰凉杯壁的雾气/烫手的纸碗） | 15% |
| 8 | **味觉瞬间** | 食物入口时的具体口感/味觉描写（烫/酥/滑/麻/辣/甜/酸/沙糯）+ 人物的微表情反应 | 7% |
| 9 | **氛围收尾** | 「氛围是/是在……」句式总结，升华到一种情感状态——孤独与满足并存、被城市温暖包裹、自由自在的微小幸福感 | 8% |

## 核心规则

1. **字数范围：580-730字**（实测原文件780-980字也可接受，但优先控制在此范围内）
2. **单行条目**：一条就是文件中的一行，不允许换行（整段文字用一个段落完成）
3. **空行分隔**：条目之间用一个空行隔开
4. **城市多样性**：尽可能覆盖不同地域城市（成都/广州/重庆/杭州/青岛/西安/南京/厦门/上海/北京/长沙/昆明等）
5. **美食对应城市特色**：成都串串香、广州糖水、重庆小面、杭州馄饨担、青岛啤酒、西安烤肉、南京鸭血粉丝汤、厦门海鲜粥、北京卤煮、长沙臭豆腐、昆明米线
6. **传统色名必须使用**：每条至少用8-12个传统色名（朱红/绀青/艾绿/苏芳/山吹/琥珀/栀子/月白/鸦青/竹青/茶色/栗色等）
7. **叙事性 > 描述性**：不是单纯的静物描写，要有时间流动感、人物的微小动作和反应、场景中发生的小事件

## 与传统7维度模板的区别

| 维度 | 02风格（都市生活/网红照） | 06风格（都市夜景/美食治愈） |
|------|--------------------------|--------------------------|
| 核心 | 人物静态肖像 | 沉浸式城市深夜叙事 |
| 风格技法 | 开篇即点明（法式复古/日杂风） | 隐藏在摄影风格层 |
| 色彩系统 | 人物穿搭为主 | 环境色板与食物色板并重 |
| 材质细节 | 服装面料精细描写 | 食物质感+锅具烟气并重 |
| 前景 | 人物 | 摊位/锅/碗/食物 |
| 出圈要素 | 风格致敬/悖论 | 不需要——叙事本身已完整 |
| 收尾 | 氛围情感 | 食物温暖+城市深夜归属感 |

## 已知陷阱

- ❌ **多行条目**：内容中含换行符会把一条拆成多行，必须确保一条为单行文本
- ❌ **字数过大**：超过800字会被用户退回重写，生成后立即验证字数
- ❌ **城市重复**：避免连续生成同城条目，交叉轮换城市
- ❌ **色名不足**：传统色名是硬性要求，每条至少8-12个
- ⚠️ **文件已有格式不一致**：原文件06.txt的实际条目长度上限约986字（虽然规范写580-730），以用户明确要求为准
