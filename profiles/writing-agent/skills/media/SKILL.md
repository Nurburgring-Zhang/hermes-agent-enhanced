# Media (媒体处理)

此分类包含多媒体内容处理相关的技能，涵盖视频转录、动图搜索、音乐识别和 YouTube 内容分析。

此分类包含以下 4 个子技能：

## 触发条件
- 用户提及此功能相关关键词时
- 遇到此领域的标准问题时
- 需要执行该领域的标准操作时


### gif-search
GIF 动图搜索工具，从各大动图平台搜索和获取动图资源。

### heartmula
音乐识别和情感分析工具，通过音频特征识别歌曲并分析情感。

### songsee
音乐可视化工具，将音频内容转换为视觉表现形式。

### youtube-content
YouTube 内容分析工具，获取视频转录、评论、元数据和内容摘要。
