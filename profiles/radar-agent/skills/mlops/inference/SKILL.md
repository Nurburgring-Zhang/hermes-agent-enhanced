# Inference (推理)

此子分类包含模型推理部署和优化相关的技能。

## 子技能

## 触发条件
- 用户提及此功能相关关键词时
- 遇到此领域的标准问题时
- 需要执行该领域的标准操作时


### gguf
GGUF 模型格式转换和量化工具，用于高效运行 LLM。

### guidance
结构化输出生成框架，通过约束解码控制语言模型输出格式。

### llama-cpp
llama.cpp 推理引擎，支持 CPU/GPU 本地高效运行量化模型。

### obliteratus
模型消融和手术工具，用于分析和修改预训练模型的结构。

### outlines
结构化文本生成库，支持正则表达式和 JSON Schema 约束输出。

### vllm
vLLM 高性能推理引擎，支持 PagedAttention 和连续批处理。

## 回滚方案
### 模型回退
1. 切换到上一个已知稳定的模型版本
2. 重新加载模型权重
3. 验证推理结果与预期一致

### 配置还原
1. 恢复配置文件到上一版本
2. 重启服务使配置生效
3. 确认服务健康检查通过
