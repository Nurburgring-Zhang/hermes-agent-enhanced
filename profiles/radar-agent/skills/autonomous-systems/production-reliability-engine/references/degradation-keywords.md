## 降级严重程度判定

| 条件 | 操作 |
|:-----|:-----|
| 0个问题 | pass — 正常通过 |
| 1-2个问题 | block_and_escalate — 拦截并生成报告 |
| 3个以上问题 | block_and_escalate — 强制拦截，需人工介入 |

## 误报防护

- 关键词匹配是精确字符串包含，不分词
- 短文本场景下不做目标关键词覆盖度检查（之前版本移除）
- 正常的高质量实现不会触发任何关键词
