此分类包含社交媒体平台集成技能，主要涵盖 X/Twitter 的交互操作。

## 触发条件
- 用户提及此功能相关关键词时
- 遇到此领域的标准问题时
- 需要执行该领域的标准操作时


### xitter
通过 x-cli 终端客户端与 X/Twitter 交互，使用官方 X API 凭证进行发帖、阅读时间线、搜索推文、点赞、转发等操作。

### xurl
通过 xurl（官方 X API CLI）与 X/Twitter 交互，支持发帖、回复、引用、搜索、时间线、私信、媒体上传和原始 v2 端点访问。
