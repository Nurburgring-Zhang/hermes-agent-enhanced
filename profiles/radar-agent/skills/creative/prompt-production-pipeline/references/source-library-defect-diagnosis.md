# AIGC 提示词源库缺陷诊断方法论

## 概述

当使用维度分离的源库文件（风格库、穿搭库、姿势库、场景库等）组合生成完整prompt时，AIGC模型会产生肢体错误（三只手、三个腿、两个头、左右衣服不一致、服装性别颠倒等）。根因不在模型本身，而在源库文件中的13类隐藏缺陷。

本文件提供系统化的诊断方法论，用于发现和修复这些缺陷。

## 参考标准文件

修复前必须准备一个无缺陷的高质量参考文件。参考文件的特征：
- 每条500-800字，6维度齐全（主体、姿势、空间、色彩、光影、氛围）
- 左右身体部位精确区分（"左手..."、"右手..."，每条约3-5次）
- 衣服与身体部位用动词绑定（"身着"、"穿着"、"佩戴"）
- 每个肢体动作有明确的空间坐标（"位于画面左侧"、"向前延伸"）

## 13类缺陷详细诊断

### P0级——直接导致畸形的严重缺陷

#### 1. 性别代词混用
**现象**：同一条prompt中"他"和"她"同时出现指代同一个人。
**诊断**：
```bash
grep -o "[他她]" 文件名.txt | sort | uniq -c | sort -rn
grep -n "他.*她\|她.*他" 文件名.txt | wc -l
```
**标准**：如果"他"和"她"的计数差小于总数×20%，且超过10%的条目同时包含两者，属于严重混用。
**修复**：统一所有条目的性别代词。每条只能用"他"或"她"中的一种，不能混用。

#### 2. 多肢体无左右区分
**现象**：只用"双手、双臂、双腿"而不分"左手、右手、左臂、右臂"。
**诊断**：
```bash
grep -o "双手\|左手\|右手" 文件名.txt | sort | uniq -c | sort -rn
```
**标准**：如果有3次以上"双手"但"左手+右手"少于"双手"次数，属于缺陷。每条至少需要1-2次左右区分。
**根因**："双手捧脸"+"右手拿着杯子"会导致AIGC在"捧脸"已占用双手的情况下再生成第三只手。
**修复**：每个对称身体部位必须标注左/右。禁止纯"双手、双臂、双腿"。

#### 3. 纯标签堆砌无动词
**现象**：整条prompt由名词+形容词堆砌，没有"身着、穿着、佩戴"等动词。
**诊断**：
```bash
grep -c "身着\|穿着\|佩戴\|搭配\|外搭\|内搭" 文件名.txt
echo "总行数: $(grep -c '' 文件名.txt)"
```
**标准**：如果少于10%的条目有绑定动词，属于严重堆砌。
**根因**：AIGC不知道衣服是穿在身上的还是摆在地上的。
**修复**：每条加入至少一个"身着、穿着、搭配、佩戴"建立绑定。

### P1级——显著影响质量的重要缺陷

#### 4. 纯风格无主体
**现象**：纯氛围或风格描述，完全不提及画面主体。
**诊断**：检查每人条是否有主体人物词。
**标准**：如果少于40%的条目包含"女性、男性、女子、男子、模特"等主体词，属于纯风格库。

#### 5. 短prompt退化
**现象**：每条少于80字，不足以约束AIGC行为。
**诊断**：检查每行长度。
**标准**：平均长度<60字，大量条目<50字。扩展至150-200字。

#### 6. 姿势与服装矛盾
**现象**：姿势要求极限姿态（跪、趴、倒立），但服装为不兼容类型（超短裙、紧身包臀裙）。
**诊断**：分别在姿势库和穿搭库中查找极限姿态词和不兼容服装词。

#### 7. 无光影色彩
**现象**：纯动作描述，没有光线方向或色彩倾向。
**诊断**：检查色彩词和光线词覆盖率。
**标准**：少于30%的条目包含光影或色彩描述。

#### 8. 纯穿搭无场景
**现象**：只有服装描述，没有环境或背景信息。
**诊断**：检查场景词覆盖率。
**标准**：少于5%的条目包含场景或环境词。

### P2级——影响整体质量的建议性问题

#### 9. 无情感氛围收尾
**现象**：描述完动作和场景后没有整体氛围收尾。
**标准**：少于20%的条目包含氛围或情感词。

#### 10. 场景重复
**现象**：同一条场景描述被重复使用30+次。
**诊断**：比较行前20字的重复率。

#### 11. 纯大师风格无人物
**现象**：描述摄影大师风格但没有具体的人物或场景。
**标准**：每条都包含"顶级摄影、获奖照片"等纯风格引导词。

#### 12. 空间定位矛盾
**现象**：不同维度库各自描述空间位置，导致冲突。
**标准**：非人体的维度库不应包含空间定位词。

## 修复流程

### 第一步：全量统计

对每个源库文件执行以下检查：

```python
checks = {
    'gender_mix': 性别代词混用率,
    'left_right': 左右区分比例,
    'tag_pile':   堆砌无动词比例,
    'no_subject': 缺主体比例,
    'too_short':  短行比例,
    'no_light':   无光影比例,
}
```

### 第二步：按优先级逐条修复

每条修复后必须满足：
- 有主体定位
- 有左右肢体区分（每条约2-4处）
- 衣服用动词绑定
- 有场景、有色彩、有光影
- 无性别代词混用
- 长度100-200字（维度库）或500-800字（完整prompt）

### 修复示例

修复前（纯堆砌）：
```
蜜茶棕及肩发半扎丸子头。奶白色方领泡泡短上衣荷叶边袖口。浅蓝牛仔A字短裙。
白色厚底老爹鞋。珍珠锁骨链叠戴金色细链。韩系甜美风。
```

修复后（加入动词绑定+场景+左右+光影）：
```
一位年轻女性将蜜茶棕发丝半扎成俏皮的丸子头。她身着奶白色方领泡泡短上衣，左袖口的荷叶边自然垂落；下搭浅蓝色高腰毛边A字短裙。脚踩白色厚底老爹鞋，颈间珍珠锁骨链与金色细链叠戴，在侧光下交错闪烁。她站在午后咖啡店门口，暖黄光线从左前方斜照，在裙摆上投下柔和阴影。整体氛围甜美而俏皮。
```
