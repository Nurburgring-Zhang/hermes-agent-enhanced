# AI文本润色 (AI Text Humanizer)

## 核心理念

AI写的东西读起来"太完美、没灵魂"。这个技能就是去掉AI味，让文章像人写的、有温度、有情绪。

## 四大润色技巧

### 1. 打破句式对称
```
❌ 首先...其次...最后...
❌ 第一...第二...第三...
✅ 直接说，不排比
✅ 长短句交错
✅ 偶尔用单句段落（短，有力）
```

### 2. 增加口语化
```
✅ "说实话"、"没想到"、"绝了"
✅ "说白了"、"怎么回事？"
✅ 适当用反问句
✅ 不要每句都完整
```

### 3. 增强情绪感染
```
❌ "这个方案效果很好"
✅ "转化率从3%飙到12%，团队都惊了"
❌ "用户体验显著提升"
✅ "用户再也不骂了——因为真的快了"
```

### 4. 具象化表达
```
❌ "效果显著提升" → ✅ "从3%飙到12%"
❌ "用户体验好"   → ✅ "3秒加载，用户不再流失"
❌ "成本降低"     → ✅ "每月省了2.3万，能多雇一个人"
```

## 格林主人特别约束

格林主人极度厌恶符号标记：
- 禁止 `**` `#` `-` `→` 等装饰符号
- 禁止 `【】` 类方括号
- 禁止编号列表格式
- 纯文字自然叙述

## 使用方式

```python
"帮我humanize这段话，去掉AI味：[内容]"
```
