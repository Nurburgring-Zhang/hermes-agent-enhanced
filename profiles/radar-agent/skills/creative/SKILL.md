# Creative (创意内容生成)

此分类包含各类创意内容生成的技能，涵盖从纯文本ASCII艺术到专业级数学动画、从知识漫画到歌曲创作的全方位创意工具。

## 触发条件
- 用户提及此功能相关关键词时
- 遇到此领域的标准问题时
- 需要执行该领域的标准操作时


### architecture-diagram
生成深色主题的 SVG 软件架构图和云基础设施图，支持微服务拓扑、云VPC网络、数据库与API分层等场景。基于 Cocoon AI 的架构图生成器。

### ascii-art
使用 pyfiglet（571种字体）、cowsay、boxes、toilet、图片转ASCII等多种工具生成 ASCII 艺术字，无需API密钥。

### ascii-video
将视频/音频/图片/生成式输入转换为彩色ASCII字符视频输出（MP4/GIF/图片序列），支持音频反应式可视化。

### baoyu-comic
创建多格知识漫画，支持多种艺术风格和色调，生成带详细面板布局的序列图像。适用于教育漫画、传记漫画、教程漫画等。

### baoyu-infographic
生成专业级信息图，支持21种布局和21种视觉风格，自动分析内容并推荐最优布局×风格组合。

### creative-ideation
通过约束条件驱动的创意生成工具。当用户说"想做点什么"或"给我一个项目想法"时，生成代码、艺术、硬件、写作等类型的项目灵感。

### excalidraw
使用 Excalidraw JSON 格式创建手绘风格的图表，支持架构图、流程图、时序图、概念图等，可在 excalidraw.com 打开。

### manim-video
使用 Manim Community Edition 创建3Blue1Brown风格的数学和技术动画视频，涵盖算法可视化、方程推导、架构图动画等。

### p5js
p5.js 交互式生成艺术管线，创建基于浏览器的创意编程作品，支持2D/3D渲染、粒子系统、着色器、像素操作、音频反应式视觉等。

### pixel-art
将图片转换为复古像素艺术，支持NES、Game Boy、PICO-8等硬件精确调色板，并可生成动画短视频。

### popular-web-designs
54套从真实网站提取的生产级设计系统模板，可生成Stripe、Linear等知名网站风格的HTML/CSS代码。

### songwriting-and-ai-music
歌曲创作技巧、AI音乐生成提示词（主攻Suno）、改编/翻唱技法、语音技巧及经验总结。

## 回滚方案
### 内容回退
1. 恢复到上一个版本的文件
2. 确认生成内容无退化
3. 必要时重启生成流程

### 恢复步骤
1. 从备份目录恢复原始文件
2. 验校内容完整性
3. 对比前后差异确认回退成功
