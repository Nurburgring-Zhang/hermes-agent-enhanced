# Cronjob模板 — 大规模prompt后台生成

## 单文件cronjob模板

```yaml
action: create
name: pg-XX   # 替换为文件名编号
deliver: local
schedule: "every 1m"
repeat: 5000
prompt: |
  【现代视觉艺术·XX.txt】输出文件路径>=11111退出。
  每次5条echo追加，600-700字wc -m验证。
  
  【核心指令】女性主体 × 风格组合。
  风格词库：美艳/冷艳/性感/魅惑/可爱/萌萌哒/青春/活力/性冷淡/优雅/慵懒/甜美/飒爽/清纯/妩媚/知性/元气/温柔/酷飒/复古
  每轮选2-3个组合。
  
  九层一段：[按九层模板结构]
  场景：[每轮轮换场景清单]
```

## 写入文件方式

```bash
echo "完整的一条提示词内容（600-700字）" >> /path/to/modern_art_prompts_XX.txt
```

## 验证字数

```bash
echo "内容" | wc -m  # 字符数
```

超700字必须缩减，低600字必须补充。

## 中断恢复

cronjob不受对话超时影响，每次独立运行。
如果某轮没产出，1分钟后的下一轮会继续。
`last_run_at` 为null表示从未运行过，需要手动 `cronjob action=run job_id=xxx`。
