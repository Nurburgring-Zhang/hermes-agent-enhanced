# PromptLibraryNode V20.5 — 22模板场景/角色变化规则

## 核心模式

**场景/角色/服装/道具发生较大变化时，不在总纲中写，而是在变化发生的分镜/页面/镜头内单起一行写：**

```
场景：新环境的描述
角色：变化后的角色特征描述
```

**同时满足：**
- 叙事功能简写，一句话说明镜头在故事中的作用
- 无**、-等markdown符号
- 变化规则在每个模板的创作原则末尾（格式：`变化规则：xxx`）
