# PromptLibraryNode V20.5 — 世界级标准升级（9种故事板format_templates）

## 升级概要

9种故事板子模式的format_template输出格式和创作原则按行业顶尖标准升级，每种模式引入该领域的核心法则。

## 电影分镜

**升级前：** 简单景别/画面/运镜/转场/时长字段，基本创作原则
**升级后：** 
- 景别字段增加焦距类型标注（标准镜头平视/长焦压缩/广角透视）
- 运镜字段增加速度形容词+角度标注
- 转场字段增加节奏感描述（顿挫/平滑/加速）
- 时长字段增加叙事节奏提示（快速剪辑1-2秒/镜，情绪沉淀3-5秒/镜）
- 叙事功能增加故事弧线位置（建立/推进/转折/高潮/余韵）
- 创作原则引入：**三镜头法则**（主镜头→过肩→单人特写）、轴线规则、视觉节奏递进序列

## 广告故事板

**升级前：** 画面/卖点/拍摄方式/品牌元素，简短原则
**升级后：**
- 卖点传达细化为理性卖点/感性卖点/品牌态度三层
- 品牌元素细化为LOGO位置/品牌色方式/Slogan时机/产品露出角度
- 创作原则引入：**AIDA法则**（Attention→Interest→Desire→Action）
- 引入产品视觉金字塔（全景20%→中景40%→特写60%→微距80%）
- 引入色彩心理学锚定（主色70%/辅色25%/点缀色5%）
- 新增铁律：画面中每个元素都在为品牌叙事服务

## 动画故事板

**升级前：** 画面/关键帧/表情/特效/时长
**升级后：**
- 动作幅度+表情变形程度指导（拉伸/挤压/弹性/惯性/夸张轨迹）
- 关键帧标注起→爆发→缓冲三阶段
- 表情/动作为三个级别（轻微/中等/极度夸张）
- 时长增加帧数指导（关键动作至少6帧）
- 创作原则引入：**迪士尼十二法则**完整列表
- 引入动作曲线（起势→发力→惯性→归位）
- 引入情绪色板（快乐=暖色圆形，悲伤=冷色锐利，愤怒=红色锯齿）
- 引入物理喜剧原则（违背程度=笑点强度）

## 漫画分镜

**升级前：** 页面布局/格1/格2描述/对话/拟声词
**升级后：**
- 格画面描述增加叙事角色（定场/推进/高潮/收尾/节奏缓冲）
- 对话框位置增加Z型阅读流约束
- 文本内容增加每格不超过20字限制
- 拟声词增加音量分级（大/中/小）+字体风格分类（爆破/尖锐/柔滑/机械）
- 创作原则引入：**Z型阅读流**引导路径
- 引入格大小=信息重要性原则
- 引入对话框避让原则（不挡面部/关键动作/不跨格边界）
- 引入翻页悬念（右页最后一格设悬念，翻页后揭晓）

## MV故事板

**升级前：** 歌词段落/画面/音乐配合/色彩调性/剪辑点
**升级后：**
- 歌词段落标注起始时间码
- 音乐配合细化到音乐结构点（前奏/主歌进鼓/预副歌/副歌爆发/桥段转调/尾奏）
- 画面与歌词关系明确三种策略：对位/反差/延展
- 创作原则引入：**音乐可视化层次**（歌词叙事→情绪渲染→节奏冲击→氛围留白）
- 引入节拍同步（镜头切换对齐强拍/弱拍/军鼓/底鼓）
- 引入歌词视觉化法则（优先隐喻/延展/反差，避免字面翻译）
- 引入色彩随音乐流动（主歌单色→副歌全色谱→桥段补色跳转）

## 教程步骤

**升级前：** 标题/画面/操作说明/重点提示/完成状态
**升级后：**
- 标题增加动词开头+不超过8字约束
- 操作增加精确到像素级的要求
- 完成状态增加多渠道反馈（截图/界面变化/声音/物理变化）
- 创作原则引入：**单步单操作**（前一步完成=后一步起始）
- 引入视觉引导系统（箭头/标注/放大镜/高亮四种元素）
- 引入容错设计（每步包含排错指南）
- 引入进度透明（第X步/共Y步+超过10步自动分组）

## 短视频分镜

**升级前：** 画面/字幕/音效/转场/心理时间
**升级后：**
- 画面强调0.5秒内阻止用户划走
- 字幕分主标题（≤6字）+副标题（≤12字），高对比配色
- 转场方式本身也是内容（花字特效=信息结束，无缝=情节连贯）
- 创作原则引入：**黄金3秒钩子**（≥2种钩子组合，不划走率>80%）
- 引入5-7秒信息节拍+15秒情绪转折
- 引入竖屏构图铁律（主体60%/底部字幕20%/顶部信息10%）
- 引入完播率设计（最后3秒互动钩子）

## 品牌故事板

**升级前：** 画面/品牌VI/情感调性/文案参考/时长
**升级后：**
- 品牌VI体现细化LOGO呈现方式（场景道具/自然光投影/角色穿戴/环境装饰）
- 情感调性明确五种反应（品牌信任/生活方式向往/身份认同/价值观共鸣/紧迫感）
- 文案参考增加声音风格标注
- 时长增加三段节奏（认知2-3秒→情感5-8秒→记忆2-3秒）
- 创作原则引入：**品牌色黄金比例**（主色60%/辅色30%/强调色10%）
- 引入LOGO露出法则（开场/高潮/结尾三节点）
- 引入品牌人格镜头语言（高端/年轻/环保各不同）
- 引入七秒品牌规则（7秒内必须能回答是什么牌子）

## 剧情分镜

**升级前：** 画面/角色情绪/戏剧冲突/镜头语言/对白
**升级后：**
- 角色情绪引入行为化描述准则（不说抽象词，写可见行为）
- 戏剧冲突明确四种来源（目标冲突/信息不对称/情绪错位/外部压力）
- 对白规则：每句话完成一个叙事任务，废对白全部删除
- 创作原则引入：**三幕冲突密度曲线**（逐幕缩短冲突间隔）
- 引入契诃夫之枪（每个出现的重要元素必须在后续被使用）
- 引入潜台词优先（对白只写冰山的八分之一）
- 引入场景节奏交替（紧张→舒缓峰谷交替）

## 修改方法

这些升级在format_templates字典中直接替换。每个模式的形成独立的块，以`"{mode_name}": (\n`开头，以`"场景切换时...\\n"\n),`结尾。替换时注意保持Python字符串拼接的正确缩进（8空格）。如果缩进乱掉，用`cp backup orig`恢复后重新执行严格的字符串替换。
