## 关键工具避坑

### patch工具转义问题
- `patch` 对Python多行字符串中的 `\"` 非常敏感
- 当 `old_string` 包含 `\"` 时会报 `Escape-drift detected`
- **缓解方案1**: 用纯文字匹配，不要包含转义符号
- **缓解方案2**: 使用 `grep -n` 找到精确行号后用 `sed -i 'N s/old/new/'`
- **缓解方案3**: 对整函数区域用 `read_file` 确认精确字符后匹配

### 搜索全局模式
修改format_template时，必须grep搜索所有模式中的相同模式。
例如改"分镜场景：（仅在场景变化时输出）"时，有4处匹配。
改"无变化写「无」"时，有6处匹配。
