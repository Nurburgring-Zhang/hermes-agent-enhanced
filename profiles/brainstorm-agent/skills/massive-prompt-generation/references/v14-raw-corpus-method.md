# v14直出法：直接使用to_chi原始语料

## 核心发现（2026-05-30）

经过v8-v13六轮迭代后，结论：
**维度库碎片拼凑永远达不到可用的质量。** 根本原因：v3维度库的条目本身就是从to_chi用正则提取的完整句子片段，不是标签。
任何用正则从完整句子里提纯标签的方法，永远有漏网之鱼。

## v14方案

直接用to_chi原始语料(`/mnt/d/ComfyUI/提示词/to_chi/`，261个文件，52万条)：
- 每条原始文本200-400字，结构完整
- 人物/服装/场景/光线自洽（人类写的）
- 无碎片、无性别断裂、无逻辑矛盾、无三手

## 过滤规则

- 去掉含比喻词(`仿佛/犹如/就像/好似/宛如/如同`)的
- 去掉含多人词(`两位/两名/情侣`)的
- 去掉非人物内容(`产品摄影/美食摄影/风景/景观`)
- 必须含人物关键词

过滤后约24万条可用(46%)。

## 质量验证

30条手工审阅：100%合格
500条抽样：0逻辑错误、0性别断裂、0三手
50000条批量：11.5秒产出

## 产出格式

每条都是一个完整的提示词段落，以"特写照片""一张肖像""一位年轻女性"等开头，以摄影参数/风格描述结尾，可直接用于AI图像生成。
